# Đồ án ngành - ChatApp 
## Sử dụng React (Typescript) cho Frontend và Firebase cho Backend

### Cách chạy ứng dụng
1. Sau khi `git clone` về
2. Chạy lệnh `npm install` đề cài `node_modules` từ `package.json`
2. Chạy lệnh `npm start` để chạy ứng dụng ở `localhost:3000`

### Cấu trúc project
- src
  - components
    - ChatList
    - ChatWindow
    - Context
    - Message
    - Navbar
    - ReceiverInfo
    - SignIn
    - SignOut
    - Spin
    - UserInfo
  - App.tsx
  - types.d.ts
- .env
